patientInfo

근기능 측정 디바이스의 임상 모니터링 웹 — Spring Boot 백엔드 + React SPA 프론트엔드
v2.0 · Phase 8 · 2026-05-18
Java 17 / Spring Boot 3.5.7 React 19 / Vite 6 / TS strict MariaDB · H2 (test) 동결 와이어 컨트랙트

이 프로젝트는 무엇인가

환자가 디바이스로 측정한 근력 시계열 데이터를 임상의가 브라우저에서 조회·비교·인쇄할 수 있게 해주는 단일-노드 웹 애플리케이션입니다. 데이터는 디바이스 → Spring Boot API → MariaDB → React SPA 의 순서로 흐르고, 모든 트래픽은 클리닉 LAN 내부에서만 발생합니다.

동결 컨트랙트: 디바이스 펌웨어와 Flutter 앱이 이미 운영 중이라 measurement_Id (대문자 I), 인제스트는 snake_case, 응답은 camelCase, 그리고 인증 없음 이라는 와이어 형태는 변경할 수 없습니다. 이 가정 아래 백엔드를 강화하고 프론트를 전면 재작성한 것이 v2.0 입니다.

주요 기능

환자 관리

목록 검색·정렬·페이지, 등록 폼 (Zod 검증)

Chart.js 그래프, 통계 카드 (피크·RFD·임펄스), CSV 내보내기

설정

테마 · 언어(ko/en) · PII 마스킹 토글 — 모두 localStorage 동기화

인쇄

전용 print stylesheet — chrome 숨김, B/W 출력 최적화

접근성

키보드, 스크린리더, prefers-reduced-motion 준수

로컬 실행

한 번만 — 로컬 dev 자격증명 파일 만들기

cp src/main/resources/application-dev.properties.example \
   src/main/resources/application-dev.properties
# 그리고 REPLACE_WITH_LOCAL_DB_USER / _PASSWORD 두 줄을 본인 로컬 MariaDB에 맞게 수정
# (이 파일은 .gitignore 되어 있어 절대 커밋되지 않음 — IMPL_SPEC §8.11)

프론트 dev 서버 (Vite 5173, HMR)

cd frontend
corepack pnpm@9.15.9 install
corepack pnpm@9.15.9 dev

백엔드 (dev 프로파일)

JAVA_HOME=$HOME/.local/jdks/jdk-17.0.13+11 \
PATH=$JAVA_HOME/bin:$PATH \
SPRING_PROFILES_ACTIVE=dev \
./gradlew bootRun -PskipFrontend=true

풀빌드 (CI 동일 게이트)

JAVA_HOME=$HOME/.local/jdks/jdk-17.0.13+11 \
PATH=$JAVA_HOME/bin:$PATH \
./gradlew build
운영 환경(=dev 프로파일 아님) 으로 실행하려면 DB_URL, DB_USERNAME, DB_PASSWORD 환경변수가 반드시 필요합니다. Phase 7 T30 이후 fallback default가 제거되어 미설정 시 부팅이 실패합니다 — 이게 정상 동작입니다.

API 요약

전체 와이어 형태는 WEB_REBUILD_PLAN.md §3 가 단일 출처입니다. 여기서는 빠른 참조용 요약만 보여줍니다.

MethodPath설명
GET/api/v1/patients환자 목록
POST/api/v1/patients환자 등록 (snake_case 입력)
POST/api/v1/measurements/start측정 세션 시작 — 응답 { measurement_Id }
POST/api/v1/measurements/{id}/data데이터 포인트 일괄 저장
POST/api/v1/measurements/{id}/stop세션 종료
GET/api/v1/patients/{patientId}/measurements환자의 세션 요약 목록
GET/api/v1/measurements/{id}/data세션의 상세 시계열

모든 에러 응답은 { status, error, timestamp } 형태의 sanitized envelope — exception 메시지는 노출되지 않습니다 (Phase 7 T31 GlobalExceptionHandler).
